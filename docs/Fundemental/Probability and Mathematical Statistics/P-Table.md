# 几种常用的概率分布表

| 分布 |参数| 概率密度函数 | 期望 | 方差 |
|:---:|:---:|:---:|:---:|:---:|
|(0-1)分布|$0<p<1$|$P\{X=k\}=p^k(1-p)^{1-k},\;k=0,1$|$p$|$p(1-p)$|
|二项分布|$n\in \mathbb{N}^*,\;0<p<1$|$P\{X=k\}=C_n^kp^k(1-p)^{n-k},\;k=0,1,...,n$|$np$|$np(1-p)$|
|几何分布|$0<p<1$|$P\{X=k\}=(1-p)^{k-1}p,\;k=1,2,...$|$\frac{1}{p}$|$\frac{1-p}{p^2}$|
